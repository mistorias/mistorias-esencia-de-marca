# Mistorias - Estrategia de Marca (v3.0)

## 1. Resumen Ejecutivo

**Mistorias** es una plataforma que **mejora la educación a través de historias**. Usa noticias relevantes de Arequipa y Perú para humanizar datos educativos, visibilizar desigualdades y convertir comprensión en acción transformadora.

**Misión**: *"Historias para transformar la educación"*

**Estrategia en 3 fases**:
1. **Fase 1**: Noticias relevantes (Arequipa/Perú)
2. **Fase 2**: Paneles estadísticos interactivos con datos públicos
3. **Fase 3**: Paneles estadísticos interactivos con datos propios recopilados

**Siempre con historias como eje central**.

**Audiencia**: Educadores y educadoras, líderes locales, tomadores y tomadoras de decisiones, ciudadanos y ciudadanas comprometidas con la educación en Arequipa y Perú.

**Equipo**: Una persona creadora (tú) + voluntarias y voluntarios que aporten via GitHub.

**Inversión**: Cero euros, excepto dominio y correo con dominio propio (~$10/año).

## 2. Análisis de Contexto

### 2.1 Oportunidad Educativa
- **Desigualdad educativa Perú**: Disparidades claras entre regiones urbanas y rurales
- **Arequipa específica**: Diferencias de acceso educativo entre municipios
- **Demanda de datos humanizados**: Las estadísticas frías no movilizan; historias sí
- **Falta de plataformas locales**: No hay periodismo educativo regional que conjugue narrativa + datos

### 2.2 Diferenciadores Competitivos
1. **Historias + Datos**: No solo números, contexto humano
2. **Foco geográfico**: Arequipa + Perú (no global genérico)
3. **Progresión estratégica**: Noticias → Datos públicos → Datos propios
4. **Secuencia transformadora**: Siente → Entiende → Actúa
5. **Diseño editorial**: Visual sofisticado para temas complejos
6. **Sostenibilidad**: Modelo de voluntariado + inteligencia artificial

### 2.3 Competidores Indirectos

| Competidor | Fortalezas | Debilidades |
|---|---|---|
| **Medios nacionales** | Alcance | Sin foco educativo |
| **Colectivos educativos** | Datos locales | Sin narrativa atractiva |
| **Universidades** | Investigación | Lenguaje académico |
| **Redes sociales locales** | Conexión | Sin rigor analítico |

**Ventaja Mistorias**: Historias que transforman datos en acción concreta.

```

### 5.2 Lenguaje Inclusivo de Género

Mistorias usa lenguaje inclusivo en todos sus espacios:

- **Sujetos plural**: "Educadores y educadoras" / "Estudiantes"
- **Sustantivos inclusivos**: "Las personas", "Ciudadanía"
- **Evitar genéricos**: No usar "el profesor" como universal
- **Alternativas neutras**: "Profesorado", "Magisterio", "Comunidad educativa"
- **Énfasis en pluralidad**: Reconocer todas las identidades y géneros

**Ejemplo**: 
```
"Estudiantes de todas las identidades de género merecen educación de calidad. 
Docentes comprometidas y comprometidos transforman vidas."
```

```
1. SIENTE: Historias que conectan emocionalmente
2. ENTIENDE: Datos, contexto, análisis claro

## 6. Estrategia de Contenido (3 Fases)

### Fase 1: Historias + Noticias (Meses 1-6)

- Noticias educativas Arequipa/Perú (2-3 por semana)
- Historias humanas detrás de datos educativos
- Recopilación de noticias vía inteligencia artificial + curaduría manual
- Secuencia: Siente → Entiende → Actúa
- Publicación: Web única centralizada

### Fase 2: Datos Públicos (Meses 7-12)

- Paneles interactivos con datos del Ministerio de Educación, Instituto Nacional de Estadística
- Visualizaciones interactivas de brechas educativas
- Historias que explican cada métrica
- Comparativas Arequipa vs Perú vs Latinoamérica
- Integración de noticias + datos en mismo análisis

### Fase 3: Datos Propios (Año 2+)

- Datos recopilados a través de formularios comunitarios
- Análisis de patrones educativos locales
- Modelos de intervención personalizados
- Impacto medible de acciones comunitarias
- Colaboraciones estratégicas (después de demostrar valor)

**Hilo conductor**: *Siempre historias primero, datos como apoyo.*

## 8. Modelo de Equipo

### 8.1 Estructura

**Fundadora/Fundador**: 
- 2-3 horas semanales
- Curaduría de noticias
- Redacción de historias principales
- Dirección editorial

**Voluntarias y Voluntarios** (via GitHub):
- Traducciones
- Historias
- Cualquier aporte que sumen

**Inteligencia Artificial**:
- Recopilación inicial de noticias
- Resúmenes analíticos
- Identificación de temas emergentes

### 8.2 Inversión Cero

```
Servidor: Vercel, Netlify u otro gratuito
Base de datos: Firebase, Supabase gratuito o autohospedado
Diseño: Figma Community gratis
Redacción: Herramientas abiertas
Costo mensual: $0 (excepto dominio + correo: ~$1/mes)
```

### 8.3 Sostenibilidad Futura

Fase 1-2: Voluntariado + inversión personal mínima

Fase 3: Evaluará modelo de sostenibilidad (donaciones, membresía, asociaciones)

## 11. Principios de Operación

### 11.1 Proceso Abierto
- Código abierto (GitHub)
- Decisiones editoriales transparentes
- Comunidad puede proponer historias
- Voluntariado valorado y reconocido

### 11.2 Calidad sobre Cantidad
- Mejor 2 historias bien investigadas que 10 superficiales
- Datos verificados con fuentes
- Edición cuidadosa, sin apuros

### 11.3 Accesibilidad
- Lenguaje claro y accesible
- Texto alternativo en imágenes
- Subtítulos en videos
- Contenido en formato texto y visual

---

## 12. Conclusión

Mistorias es **único** porque:

1. **Propósito transformador** (educación) + **historias humanas** (narrativa) con **lenguaje inclusivo**
2. **Datos progresivos** (noticias → públicos → propios) + **acción concreta**
3. **Modelo sostenible**: Una creadora + voluntariado + inteligencia artificial
4. **Foco geográfico** (Arequipa/Perú) + **impacto medible y humilde**
5. **Proceso abierto**: Código, decisiones y comunidad transparentes

La estrategia de marca posiciona a Mistorias como **la plataforma educativa peruana que transforma comprensión en acción**: profundamente enraizada, visualmente sofisticada, accesible, inclusiva y sostenible.
